# Crypto Dca

Crypto bot that apply DCA (dollar cost averaging) strategy depending on your configuration.

Supported exchanges:

- [Binance](https://www.binance.com/)

Supported deployment modes:

- [AWS](https://aws.amazon.com/)

## Getting started

### Requirements

- NodeJS 14.x
- Yarn 1.x
- AWS SAM 1.x
